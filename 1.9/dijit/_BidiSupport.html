dijit/_BidiSupport (version 1.9)

Deprecated module for enabling textdir support in the dijit widgets. New code should just define has("dojo-bidi") to return true, rather than manually requiring this module.

Usage:

_BidiSupport();

See the dijit/_BidiSupport reference documentation for more information.
