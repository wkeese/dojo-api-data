dojox/app/controllers/History (version 1.9)

dojox/app/Controller

Usage:

var foo = new History();
Defined by dojox/app/controllers/History

Bind "app-domNode" event on dojox/app application instance. Bind "startTransition" event on dojox/app application domNode. Bind "popstate" event on window object.

See the dojox/app/controllers/History reference documentation for more information.

Property Summary

  • _currentPosition Persistent variable which indicates the current position/index in the history (so as to be able to figure out whether the popState event was triggerd by a backward or forward action).
  • currentState Current state

Method Summary

  • bind(evented, event, handler) Bind event on dojo/Evented instance, document, domNode or window.
  • constructor() Bind "app-domNode" event on dojox/app application instance.
  • unbind(evented, event) remove a binded event signal.

Event Summary

  • onDomNodeChange(evt)
  • onPopState(evt) Response to dojox/app "popstate" event.
  • onStartTransition(evt) Response to dojox/app "startTransition" event.

Examples

Example 1

Use "dojox/mobile/TransitionEvent" to trigger "startTransition" event, and this function will response the event. For example: 

var transOpts = {
    title:"List",
    target:"items,list",
    url: "#items,list",
    params: {"param1":"p1value"}
};
new TransitionEvent(domNode, transOpts, e).dispatch();
