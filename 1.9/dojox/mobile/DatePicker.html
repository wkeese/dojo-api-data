dojox/mobile/DatePicker (version 1.9)

A wrapper widget around SpinWheelDatePicker or ValuePickerDatePicker. It should be used with the automatic theme loader, dojox/mobile/deviceTheme. Returns ValuePickerDatePicker when the current theme is "android" or "holodark". Returns SpinWheelDatePicker otherwise.

See the dojox/mobile/DatePicker reference documentation for more information.
