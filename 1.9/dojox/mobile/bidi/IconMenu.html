dojox/mobile/bidi/IconMenu (version 1.9)

Support for control over text direction for mobile IconMenu widget, using Unicode Control Characters to control text direction.

Implementation for text direction, textDir is set to MenuItems. This class should not be used directly. Mobile IconMenu widget loads this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.

See the dojox/mobile/bidi/IconMenu reference documentation for more information.
