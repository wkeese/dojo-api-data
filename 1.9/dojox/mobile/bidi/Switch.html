dojox/mobile/bidi/Switch (version 1.9)

Bidi support for mobile Switch widget, using Unicode Control Characters to control text direction.

Implementation for text direction support for LeftLabel and RightLabel. This class should not be used directly. Mobile Switch widget loads this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.

See the dojox/mobile/bidi/Switch reference documentation for more information.
