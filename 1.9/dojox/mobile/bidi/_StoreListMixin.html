dojox/mobile/bidi/_StoreListMixin (version 1.9)

Support for control over text direction for mobile _StoreListMixin and _DataListMixin.

Property textDir is set to created ListItem. This class should not be used directly. Mobile _StoreListMixin and _DataListMixin load this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.

See the dojox/mobile/bidi/_StoreListMixin reference documentation for more information.
