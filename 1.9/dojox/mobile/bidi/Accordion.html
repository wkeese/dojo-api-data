dojox/mobile/bidi/Accordion (version 1.9)

Support for control over text direction for mobile Accordion widget, using Unicode Control Characters to control text direction.

Implementation for text direction support for Label. This class should not be used directly. Mobile Accordion widget loads this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.

See the dojox/mobile/bidi/Accordion reference documentation for more information.
