dojox/mobile/bidi/IconItem (version 1.9)

Support for control over text direction for mobile IconItem widget, using Unicode Control Characters to control text direction.

Implementation for text direction for Label. This class should not be used directly. Mobile IconItem widget loads this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.

See the dojox/mobile/bidi/IconItem reference documentation for more information.
