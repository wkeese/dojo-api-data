dojox/drawing/library/icons (version 1.9)

A collection of icon Stencils for use with the buttons in the ui/Toolbar.

Each icon name corresponds to a Stencil or a Plugin. One can be inserted into a button by adding the property: 'icon', which points to one of these Stencil objects.
