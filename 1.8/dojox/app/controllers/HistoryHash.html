Examples

Example 1

Use "dojox/mobile/TransitionEvent" to trigger "startTransition" event, and this function will response the event. For example: 

var transOpts = {
    title:"List",
    target:"items,list",
    url: "#items,list"
    params: {"param1":"p1value"}
};
new TransitionEvent(domNode, transOpts, e).dispatch();
