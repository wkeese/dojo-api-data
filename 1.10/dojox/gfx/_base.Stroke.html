dojox/gfx/_base.Stroke (version 1.10)

A stroke defines stylistic properties that are used when drawing a path.

See the dojox/gfx/_base.Stroke reference documentation for more information.

Property Summary

  • cap The endcap style of the path.
  • color The color of the stroke, default value 'black'.
  • join The join style to use when combining path segments.
  • style The style of the stroke, one of 'solid', ... .
  • width The width of a stroke, default value 1.
