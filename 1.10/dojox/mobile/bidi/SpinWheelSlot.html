dojox/mobile/bidi/SpinWheelSlot (version 1.10)

Support for control over text direction for mobile SpinWheelSlot widget, using Unicode Control Characters to control text direction.

This class should not be used directly. Mobile SpinWheelSlot widget loads this module when user sets "has: {'dojo-bidi': true }" in data-dojo-config.
